# 📁 GitHub CLI

The GitHub CLI (`gh`) lets you interact with GitHub directly from your terminal — no browser needed for most tasks.

## 📦 Installation

```bash
# Windows (winget)
winget install GitHub.cli

# macOS
brew install gh

# Ubuntu/Debian
sudo apt install gh
```

Verify:
```bash
gh --version
```
