# ⚙️ 01 — Setup

Commands for configuring Git and initializing or cloning repositories.

> Start here if you're setting up Git for the first time.

---

## 📄 Commands in This Folder

| Command | File | Description |
|---|---|---|
| `git config` | [config.md](./config.md) | Set username, email, editor, and aliases |
| `git init` | [init.md](./init.md) | Initialize a new Git repository |
| `git clone` | [clone.md](./clone.md) | Copy a remote repo to your local machine |
| `git help` | [help.md](./help.md) | Get documentation for any Git command |

---

## 🔰 Recommended Order for Beginners

1. [config.md](./config.md) — set your identity first
2. [init.md](./init.md) — start a new project
3. [clone.md](./clone.md) — copy an existing project
4. [help.md](./help.md) — get help anytime
