## 📌 What's Inside

A quick-reference collection of the most useful Git and GitHub commands — from everyday basics to intermediate workflows. Each command includes a brief explanation and practical example so you can understand *what* it does and *when* to use it.

## 🚀 Topics Covered

- Setting up Git (`config`, `init`, `clone`)
- Staging & committing (`add`, `commit`, `status`, `log`)
- Branching & merging (`branch`, `checkout`, `merge`, `rebase`)
- Remote workflows (`remote`, `push`, `pull`, `fetch`)
- Undoing changes (`reset`, `revert`, `stash`)
- GitHub-specific workflows (forks, pull requests, issues)
