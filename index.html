Setup & Configuration
Initialize repos and configure Git

git config

Essential

Set or view configuration variables like username, email, and editor.

git config --global user.name "Your Name"
git config --global user.email "email@example.com"

💡 Use --global for all projects, --local for current repo only

git init

Essential

Initialize a new Git repository in the current directory.

git init

Creates a .git directory with repository structure

git clone

Essential

Clone a remote repository to your local machine.

git clone https://github.com/user/repo.git

Creates a complete copy with full history

git help

Utility

Display help information for Git commands.

git help <command>

Example: git help commit

Staging & Committing
Stage changes and create commits

git add

Essential

Stage changes for the next commit.

git add .
git add filename.js

Use . for all changes, or specify individual files

git commit

Essential

Save staged changes with a descriptive message.

git commit -m "Add new feature"
git commit -am "Update file"

Flag -a skips staging for tracked files

git status

Utility

View the status of your working directory.

git status

Shows modified, staged, and untracked files

git diff

Utility

Show line-by-line changes between commits or working tree.

git diff
git diff --staged

Use --staged to see staged changes

git mv

Utility

Move or rename tracked files.

git mv old_name.js new_name.js

Automatically stages the rename operation

git rm

Caution

Remove files from tracking and working directory.

git rm filename.js
git rm --cached filename.js

Use --cached to keep the file locally

Branching & Merging
Create and merge branches

git branch

Essential

Create, list, or delete branches.

git branch
git branch feature-name
git branch -d branch-name

git checkout

Essential

Switch to a different branch or restore files.

git checkout branch-name
git checkout -b new-branch

Flag -b creates and switches to a new branch

git switch

Modern

Modern way to switch branches (Git 2.23+).

git switch branch-name
git switch -c new-branch

Clearer syntax than checkout

git merge

Essential

Combine two branches together.

git merge feature-branch
git merge --no-ff feature-branch

Flag --no-ff creates a merge commit

git rebase

Advanced

Rewrite history to apply commits on top of another branch.

git rebase main
git rebase -i HEAD~3

Interactive mode allows editing commits

git cherry-pick

Advanced

Apply a specific commit from another branch.

git cherry-pick commit-hash

Useful for backporting fixes

git tag

Release

Mark release points in the repository history.

git tag v1.0.0
git push origin v1.0.0

Use semantic versioning: major.minor.patch

Remote Workflows
Collaborate with remote repositories

git remote

Essential

Manage remote repository connections.

git remote -v
git remote add origin https://github.com/user/repo.git

git push

Essential

Upload commits to a remote repository.

git push origin main
git push -u origin branch-name

Flag -u sets upstream tracking

git pull

Essential

Download and merge changes from remote.

git pull origin main
git pull --rebase

Combines fetch and merge operations

git fetch

Safe

Download changes from remote without merging.

git fetch origin

Safe way to review changes before merging

git submodule

Advanced

Manage nested repositories within a project.

git submodule add URL path

Useful for managing dependencies

Undoing Changes
Undo commits and discard changes

git reset

Caution

Undo commits or unstage files.

git reset HEAD~1
git reset --soft HEAD~1

Use --soft to keep changes staged

git revert

Safe

Safely undo a commit by creating a new one.

git revert commit-hash

Preferred for pushed commits

git restore

Modern

Discard changes in working directory.

git restore filename.js
git restore --staged filename.js

Modern replacement for checkout

git stash

Utility

Temporarily save uncommitted changes.

git stash
git stash pop

Great for switching branches quickly

git clean

Caution

Remove untracked files from working directory.

git clean -fd

Flags: -f force, -d directories

Inspection & History
View history and inspect changes

git log

Essential

View commit history.

git log --oneline
git log --graph --all --decorate

Visualize commit graph with --graph

git show

Utility

Display changes from a specific commit.

git show commit-hash

Shows commit message and diff

git blame

Investigation

See who changed each line in a file.

git blame filename.js

Useful for tracking down changes

git bisect

Advanced

Find the commit that introduced a bug.

git bisect start

Binary search through commits

git shortlog

Report

Summarize commit history by author.

git shortlog -sn

Useful for generating changelogs

Advanced Operations
Power user commands

git reflog

Recovery

Recover lost commits using the reference logs.

git reflog

Lifesaver for recovering lost work

git worktree

Advanced

Work on multiple branches simultaneously.

git worktree add ../branch-name

Creates a separate working directory

git archive

Export

Export repository as archive (zip or tar).

git archive --format zip HEAD > repo.zip

Creates snapshot without .git folder

git gc

Maintenance

Clean up and optimize repository.

git gc

Reduces disk space and improves performance

git hooks

Automation

Automate actions on Git events.

nano .git/hooks/pre-commit

Located in .git/hooks/ directory

Patching
Create and apply patches

git apply

Patch

Apply a patch file to your working directory.

git apply patch-file.patch

Does not commit changes

git format-patch

Patch

Create patch files from commits.

git format-patch -n main

Useful for email-based contributions

git am

Patch

Apply patches preserving author information.

git am patch-file.patch

Better than apply for maintaining history
