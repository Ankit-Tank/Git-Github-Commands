Setup & Configuration

Initialize repositories and configure Git settings

git config

Essential

Set username, email, editor, and other Git settings

git config --global user.name "Your Name"
git config --global user.email "email@example.com"
git config --list

git init

Essential

Initialize a new Git repository

git init
git init [project-name]

git clone

Essential

Copy a remote repository to your local machine

git clone https://github.com/user/repo.git
git clone [url] [directory]

git help

Helpful

Get help documentation for any Git command

git help [command]
git help commit

Staging & Committing

Stage changes and create meaningful commit history

git add

Essential

Stage changes for the next commit

git add .
git add [filename]
git add -A

git commit

Essential

Save staged changes to repository history

git commit -m "message"
git commit -am "message"
git commit --amend

git status

Essential

Check working tree and staging area status

git status
git status -s

git diff

Intermediate

View line-by-line differences between files

git diff
git diff --staged
git diff HEAD~1

git mv

Intermediate

Move or rename tracked files

git mv [old-path] [new-path]
git mv file.js components/file.js

git rm

Intermediate

Remove files from tracking

git rm [filename]
git rm --cached [filename]

Branching & Merging

Create branches and merge changes efficiently

git branch

Essential

Create, list, and delete branches

git branch
git branch [branch-name]
git branch -d [branch-name]

git checkout

Essential

Switch branches or restore files

git checkout [branch-name]
git checkout -b [new-branch]
git checkout -- [filename]

git switch

Essential

Modern way to switch branches (newer Git versions)

git switch [branch-name]
git switch -c [new-branch]

git merge

Intermediate

Combine branch histories

git merge [branch-name]
git merge --no-ff [branch-name]
git merge --squash [branch-name]

git rebase

Advanced

Rewrite history and create linear commits

git rebase [branch-name]
git rebase -i HEAD~3
git rebase --continue

git cherry-pick

Intermediate

Apply specific commits to current branch

git cherry-pick [commit-hash]
git cherry-pick [start]..[end]

git tag

Intermediate

Mark release versions and important milestones

git tag [tag-name]
git tag -a [tag-name] -m "message"
git push origin [tag-name]

Work with remote repositories and collaborate

git remote

Essential

Manage remote repository connections

git remote -v
git remote add [name] [url]
git remote remove [name]

git push

Essential

Upload commits to remote repository

git push origin [branch]
git push -u origin [branch]
git push origin --delete [branch]

git pull

Essential

Download and merge remote changes

git pull
git pull origin [branch]
git pull --rebase

git fetch

Essential

Download remote changes without merging

git fetch
git fetch origin
git fetch --all

git submodule

Advanced

Manage nested repositories

git submodule add [url] [path]
git submodule update --init

Undoing Changes

Fix mistakes and undo unwanted changes

git reset

Intermediate

Undo commits or unstage files

git reset [filename]
git reset HEAD~1
git reset --hard HEAD~1

git revert

Intermediate

Safely undo a pushed commit by creating a new one

git revert [commit-hash]
git revert HEAD

git restore

Essential

Discard changes in working directory

git restore [filename]
git restore --staged [filename]

git stash

Intermediate

Temporarily save uncommitted changes

git stash
git stash pop
git stash list

git clean

Intermediate

Remove untracked files from working directory

git clean -fd
git clean -X

Inspection & History

Analyze commits and track changes

git log

Essential

View commit history

git log
git log --oneline
git log --graph --all --decorate

git show

Intermediate

git show [commit-hash]

git blame

Intermediate

See who changed each line and when

git blame [filename]

git bisect

Advanced

Find the commit that introduced a bug

git bisect start

git bisect good [old-commit]

git shortlog

Intermediate

Summarized commit history by author

git shortlog

git gc
