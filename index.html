01 — Setup
Configure Git and initialize repositories

git config

setup

Set your username, email, default editor and aliases globally or per-repo.

git config --global user.name "Name"

git init

setup

Initialize a new local Git repository in the current directory.

git init

git clone

setup

Copy a remote repository to your local machine.

git clone <url>

git help

setup

Get detailed documentation for any Git command.

git help <command>
📦
02 — Staging & Committing
Track, stage and save changes

git add

staging

Stage changes to be included in the next commit.

git add .

git commit

staging

Save a snapshot of staged changes to the repository history.

git commit -m "message"

git status

staging

Show the state of the working directory and staging area.

git status

git diff

staging

Show line-by-line differences between files or commits.

git diff

git mv

staging

Move or rename a tracked file and stage the change automatically.

git mv old.txt new.txt

git rm

staging

Remove a file from the repository and stage the deletion.

git rm <file>
🌿
03 — Branching & Merging
Create, switch, merge and manage branches

git branch

branching

Create, list, rename and delete branches in the repository.

git branch <name>

git checkout

branching

Switch branches or restore working tree files to a previous state.

git checkout <branch>

git switch

branching

Modern, safer way to switch between branches (Git 2.23+).

git switch <branch>

git merge

branching

Combine changes from another branch into the current branch.

git merge <branch>

git rebase

advanced

Reapply commits on top of another base, creating a linear history.

git rebase <branch>

git cherry-pick

advanced

Apply a specific commit from another branch onto the current branch.

git cherry-pick <hash>

git tag

branching

Mark specific commits as release versions or important milestones.

git tag v1.0.0
🌐
04 — Remote Workflows
Connect and sync with remote repositories

git remote

remote

Manage connections to remote repositories (add, remove, rename).

git remote add origin <url>

git push

remote

Upload local commits and branches to a remote repository.

git push origin main

git pull

remote

Fetch and automatically merge changes from the remote branch.

git pull origin main

git fetch

remote

Download changes from remote without merging into your branch.

git fetch origin

git submodule

advanced

Manage nested repositories inside a parent repository.

git submodule add <url>
↩️
05 — Undoing Changes
Reverse, reset and recover mistakes

git reset

undo

Undo commits or unstage files. Use --soft, --mixed or --hard carefully.

git reset --soft HEAD~1

git revert

safe

Safely undo a pushed commit by creating a new reverse commit.

git revert <hash>

git restore

undo

Discard changes in the working directory or unstage a file.

git restore <file>

git stash

utility

Temporarily save uncommitted changes to work on something else.

git stash

git clean

undo

Remove untracked files and directories from the working tree.

git clean -fd
🔍
06 — Inspection & History
Explore logs, blame and find bugs

git log

history

View the full commit history with author, date and message.

git log --oneline --graph

git show

history

Inspect the details and changes introduced by a specific commit.

git show <hash>

git blame

history

Show who last modified each line of a file and when.

git blame <file>

git bisect

history

Use binary search to find the commit that introduced a bug.

git bisect start

git shortlog

history

Show a summarized commit history grouped by author and count.

git shortlog -sn
⚡
07 — Advanced
Power tools for complex workflows

git reflog

advanced

View a log of all HEAD movements to recover lost commits.

git reflog

git worktree

advanced

Check out multiple branches simultaneously in separate directories.

git worktree add <path> <branch>

git archive

advanced

Export the repository contents as a zip or tar archive file.

git archive --format=zip HEAD > out.zip

git gc

advanced

Run garbage collection to clean up and optimize the repository.

git gc --prune=now

git hooks

advanced

Scripts that run automatically on Git events like commit or push.

.git/hooks/pre-commit

08 — Patching
Create and apply patch files

git apply

patching

Apply a patch file to the working directory or index.

git apply <patch-file>

git format-patch

patching

Create patch files from one or more commits for sharing via email.

git format-patch HEAD~1

git am

patching

Apply a patch while preserving the original author information.

git am <patch-file>
